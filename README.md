# book-quiz

## Build & Run Commands
- Build: `./gradlew build`
- Run: `./gradlew bootRun`
- Test: `./gradlew test`
- Run single test: `./gradlew test --tests "com.example.book_quiz.SpecificTest"`
- Run test method: `./gradlew test --tests "com.example.book_quiz.TestClass.testMethod"`
- Database: `docker-compose up` (starts PostgreSQL)
